1. Перейдите в папку backend:
   cd /Users/tlgn/CascadeProjects/mes_systems/15nov/my_fastapi_project/backend

2. Активируйте виртуальное окружение:
   source venv/bin/activate

   uvicorn main:app --reload --port 8000

                        РЕШЕНИЕ ПРОБЛЕМ
═══════════════════════════════════════════════════════════════════════════════

❌ Backend не запускается (Port 8000 in use):
   lsof -ti:8000 | xargs kill -9
   
   Или используйте другой порт:
   uvicorn main:app --reload --port 8001

❌ Frontend не запускается (Port 3000 in use):
   PORT=3001 npm run dev

❌ Ошибка "No module named 'fastapi'":
   source venv/bin/activate
   pip install -r requirements.txt

❌ AI функции не работают:
   - Проверьте backend/.env
   - Убедитесь, что OPENAI_API_KEY заполнен
   - Проверьте: http://localhost:8000/chatgpt/health

❌ CORS ошибка (Access blocked):
   Убедитесь, что backend запущен на http://localhost:8000
